## O que é compilador e interpretador?

# Compilador:
É um software que transforma o nosso código em código de máquina (executável, .ex ou binário)
e executa esse código.


Vantagem: O código-fonte está no formato binário, então é difícl ler a implementação de alguma funcionalidade do programa.

# Interpretador:
É um software que lê e executa o nosso programa.

Vantagem: A execução do código é rápia, pois não  precisa compiladar o programa.

Desvantagem: A execução do código-fonte é lenta, 
pois o processo de analisar e executar são feitas no momento de executar o nosso programa.


# Interpretador: Analisa e executa
# Compilador: Executa (binário)


código-fonte -> interpretador -> executa
código-fonte -> compilador -> binário -> executa

Código-fonte foi convertidor por software (compilador) pra ser executável.
